Making your own theme with Renard should be easy. First, create a new directory in wp-content/themes; let us name it, foobar. In the foobar directory, create an index.php file that contains your theme. See the list of available tags to see what you can do to customize the theme.

Now, you should use the <$Load*$> tags to help keep your source clean; for example, use <$LoadSidebar$> and have your sidebar code in sidebar.php. The only only thing you really need in index.php, however, is the loop: <$WPEntriesLoop$> and family.

You also need to create a style.css file that contains information about your theme. See the existing themes' style.css files for an example.

More information about making themes is available on the official theme development page.
